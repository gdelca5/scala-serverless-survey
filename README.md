# Scala Serverless Frameworks Survey

## Quick Start

```
sbt assembly
```

...and deploy the JAR to your Lambda environment.

## Using Serverless framework

Check it out from [Serverless framework](https://serverless.com/)

```
sbt assembly
sls deploy
```
